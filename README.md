# The Cuddle Village Inc. – E-Commerce Platform

The Cuddle Village Inc. is a premium baby and parenting e-commerce platform designed to support parents, mothers-to-be, and families through a combination of quality baby products, educational resources, and expert parenting support services.

## The platform combines:

A modern baby products e-commerce store

A children’s book club and learning corner

Parenting support resources

Professional guidance services (nutritionists, pediatricians, OB/GYN support)

## Installation

Clone the Repository

git clone https://github.com/alison-chelimo/the-cuddle-village.git

cd cuddle-village

## Install Dependencies

### Frontend

cd cuddle-village-frontend

npm install

### Backend

cd cuddle-village-backend

npm install

## Environment Variables

Create a .env file in the backend directory:

PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_jwt_secret

PAYSTACK_SECRET_KEY=your_paystack_secret

PAYSTACK_PUBLIC_KEY=your_paystack_public


## Running the Application

### Start Backend

cd cuddle-village-backend

npm run dev

### Start Frontend

cd cuddle-village-frontend

npm start
